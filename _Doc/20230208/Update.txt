 
V61
- Comm.Report เซลล์เห็นเฉพาะคอมของตัวเอง
- ลักษณะงานเพิ่ม 1. ส่งของ,2. ส่งใบเสร็จ
- หน้าตัดชำระ ไม่บังคับเลือกลูกค้า
- หน้าทำรายการเพิ่ม tick ว่าจะดึงไป mass ไหม
- หน้าข้อมูลบริษัท เพิ่มการตั้งเวลาแจ้งเตือนการบันทึกส่งของ กรณีไม่มีการ Assign 9.30 , ไม่สำเร็จ 17.30
- เพิ่มเมนูเพื่อกดดูแบบ Manual  แจ้งเตือนการบันทึกส่งของ รณีไม่มีการ Assign และ ไม่สำเร็จ 

V62
- เพิ่มการใส่ข้อมูลราคาขายจริง
- เพิ่มข้อมูล sku ที่หน้าสินค้า
- หน้ากำหนดราคา เพิ่มการ Import ราคาจาก sku
- เพิ่มหน้ากำหนดจำนวนวันประกัน แสดงที่หน้า สินค้า / ระบบแจ้งราคา /กำหนดราคาสินค้า
- เพิ่มรายละเอียดโปรโมชัน ในหน้าแจ้งราคา
- แก้การคำนวนคอมมิชชัน

V64
-กำหนดราคาจากบริษัท แม่ เวลาลูก2 มาเปิดใบสั่งขายให้ดึงราคาจากแจ้ง จากแม่ หรือเกณฑ์ราคาแม่
-ประวัติซื้อขายถ้าบริษัทไหน เป็นบริษัทแม่/ลูกกันดึงมาด้วย
-โปรโมชั่นซื้อ/ขาย ค้นหาลูกค้า เพิ่ม column กลุ่มลูกค้าให้ค้นหาและเลือกทั้งหมดได้ และ clone โปรเดิมได้
-หน้าจอ Import สินค้า เพิ่มข้อมูล น้ำหนัก ขนาด รุ่น สี sku
-หน้าจอ Import สินค้า เพิ่มให้เลือก export ข้อมูล


- เพิ่มหน้า เอกสารอื่นๆ เหมือน บันทึกเงินสดย่อย
